Project Overview:
Quotify is a simple and elegant quote generator app built using Flutter.
It allows users to generate motivational and inspiring quotes instantly with a single tap.
The app focuses on a clean UI and smooth user experience, making it perfect for daily inspiration.


Features :
 Generate random quotes instantly
 Clean and minimal user interface
 Beautiful background with overlay design
 Fast and lightweight
 Fully responsive layout


Run the Project:
flutter pub get
flutter run
